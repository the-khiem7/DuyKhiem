## Overview

CellphoneZ is a mobile commerce application covering the customer journey from browsing to checkout. The project helped strengthen my understanding of consumer-facing product flow and the backend support needed behind transactional features.

## Scope

- Product browsing and cart behavior
- Checkout and payment flow
- Store locator capability
- Admin-facing support features

## What It Added To My Skill Set

This project was useful because it connected product UX decisions to backend and data requirements in a commerce setting.
